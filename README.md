GPSTracker-1.0.0 Minecraft plugin

Это плагин для майнкрафта, который нужен для того чтобы отслеживать ваших друзей с помощью комманд /gps add <ник> и /gps track <ник>. Доступные для работы плагина версии майнкрафта 1.20.x - 1.21.x.

Команды плагина:
/gps help - список комманд;
/gps add <ник> - добавить друга;
/gps track <ник> - отслеживать друга;
/gps accept - принять запрос в друзья для вашего отслеживания;
/gps deny - отказать в запросе в друзья для вашего отслеживания;
/gps toggle - Включить/выключить навигацию;
/gps list - Список друзей с координатами.
